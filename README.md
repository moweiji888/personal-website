# 个人网站项目

一个现代化的静态个人网站，包含作品集、博客系统、联系表单和暗色模式。

## 🚀 功能特性

- ✨ **响应式设计** - 完全适配所有设备
- 🌓 **暗色模式** - 支持亮色/暗色主题切换
- 📝 **博客系统** - 发布和管理博客文章
- 📧 **联系表单** - 收集访客消息
- 🎨 **现代设计** - 优雅的界面设计和动画效果
- ⚡ **高性能** - 纯静态网站，快速加载
- 📱 **移动优先** - 移动设备优先的开发方式

## 📂 项目结构

```
personal-website/
├── index.html                 # 首页
├── about.html                 # 关于我页面
├── blog.html                  # 博客列表页面
├── contact.html               # 联系我页面
├── assets/
│   ├── styles.css             # 全局样式表
│   └── script.js              # 全局脚本文件
├── blog/
│   ├── hello-world.html       # 第一篇博客
│   ├── web-dev-tips.html      # 第二篇博客
│   ├── css-tips.html          # 第三篇博客
│   └── javascript-fundamentals.html  # 第四篇博客
└── README.md                  # 项目说明文件
```

## 🛠️ 技术栈

- **HTML5** - 结构和语义
- **CSS3** - 样式和响应式布局
- **JavaScript** - 交互功能和动画
- **GitHub Pages** - 免费托管

## 🌓 暗色模式

网站自动检测系统主题偏好，并支持手动切换：
- 点击导航栏的主题按钮 (🌙/☀️) 切换
- 主题偏好保存在本地存储中
- 支持键盘快捷键：Alt + T

## 📱 响应式设计

网站采用移动优先的设计方式：
- 移动设备 (< 768px)：单列布局
- 平板设备 (768px - 1024px)：双列布局
- 桌面设备 (> 1024px)：完整布局

## 🚀 快速开始

### 部署到 GitHub Pages

1. 确保代码已推送到 GitHub
2. 在仓库设置中启用 GitHub Pages：
   - 进入 Settings > Pages
   - 选择 Source 为 `main` 分支
   - 点击 Save
3. 等待几分钟后，你的网站将在以下地址可访问：
```
https://moweiji888.github.io/personal-website/
```

## ✏️ 自定义网站

### 修改个人信息

编辑各个 HTML 文件中的内容：
- 更新名字和标题
- 修改项目描述
- 添加真实的社交媒体链接
- 更新联系邮箱

### 添加新的博客文章

1. 在 `blog/` 目录下创建新的 HTML 文件
2. 使用现有文章作为模板
3. 在 `blog.html` 中添加文章链接

### 自定义样式

编辑 `assets/styles.css`：
- 修改 CSS 变量定义颜色和字体
- 调整 `:root` 中的颜色值
- 修改响应式断点
